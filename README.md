# Big Data Analytics Learning Platform

A comprehensive, interactive study platform designed for students to master Big Data Analytics concepts (Units 1-6). Built with a focus on examination performance, detailed conceptual understanding, and viva preparation.

## 🚀 Features

- **Production URL:** [https://big-data-analytics-rtu.vercel.app/](https://big-data-analytics-rtu.vercel.app/)
- **Structured Curriculum:** Detailed notes for all 6 units, from Big Data basics to advanced Hive querying.
- **Viva Readiness:** Integrated "Viva-Ready" answers and "Memory Tricks" for every key topic.
- **Interactive Diagrams:** Custom SVG schematics for GFS, HDFS, MapReduce, Hive Architecture, and more.
- **Optimized UX:**
  - **Auto-Scroll:** Content area automatically scrolls to the top when navigating between topics or units.
  - **Keyboard Navigation:** 
    - `Left/Right Arrows`: Switch between previous and next entries.
    - `Up/Down Arrows`: Smoothly scroll within the current topic.
- **Clean Aesthetic:** Editorial-grade typography and layout for distraction-free learning.
- **Searchable Content:** Quickly find specific topics across the entire syllabus.

## 📚 Syllabus Coverage

1. **Unit 1: Introduction to Big Data** - Definition, 3 V's, Challenges, and Data Types.
2. **Unit 2: Working with Big Data** - GFS, HDFS Internals, Building Blocks, and Cluster Modes.
3. **Unit 3: Writing MapReduce Programs** - Programming Model, Job Flow, and Optimization.
4. **Unit 4: Hadoop I/O** - Writable Interface, Serialization, and Raw Comparators.
5. **Unit 5: Apache Pig** - Pig Latin, Architecture, ABC Operators, and Modes.
6. **Unit 6: Apache Hive** - HiveQL, Metastore, Managed vs. External Tables, and Analysis.

## 🛠️ Tech Stack

- **Framework:** React 18+ (Vite)
- **Styling:** Tailwind CSS
- **Icons:** Lucide React
- **Animations:** Motion (Framer Motion)
- **Markdown:** React Markdown
- **Components:** Headless UI patterns for accessible interaction

## 📖 How to Use

1. **Navigate:** Use the sidebar to select a specific Unit.
2. **Read:** Use the content area to study. Each topic includes diagrams and exam-focused summaries.
3. **Shortcuts:**
   - Use `Left/Right` keys to flip through topics like a book.
   - Use `Up/Down` keys for a smooth reading experience.
4. **Search:** Use the search bar at the top of the sidebar to jump to specific concepts.
